# Starlight Documentation Framework

Starlight is the official Astro documentation framework — a full-featured docs site builder on top of Astro.

## Installation

```bash
# New Starlight project
npm create astro@latest -- --template starlight

# Add to existing Astro project
npx astro add starlight
```

## Full astro.config.mjs Reference

```typescript
import { defineConfig } from 'astro/config';
import starlight from '@astrojs/starlight';

export default defineConfig({
  integrations: [
    starlight({
      title: 'My Docs',
      description: 'Documentation for My Project',
      logo: {
        light: './src/assets/logo-light.svg',
        dark: './src/assets/logo-dark.svg',
        replacesTitle: true,
      },
      social: {
        github: 'https://github.com/org/repo',
        discord: 'https://discord.gg/...',
        twitter: 'https://twitter.com/...',
      },
      editLink: {
        baseUrl: 'https://github.com/org/repo/edit/main/docs/',
      },
      lastUpdated: true,
      pagination: true,
      tableOfContents: { minHeadingLevel: 2, maxHeadingLevel: 4 },
      favicon: '/favicon.svg',
      head: [
        // Custom head tags
        { tag: 'script', attrs: { src: '/analytics.js', defer: true } },
      ],
    }),
  ],
});
```

## Sidebar Configuration

```typescript
starlight({
  sidebar: [
    // Auto-generate from directory
    { label: 'Guides', autogenerate: { directory: 'guides' } },

    // Collapsed section
    {
      label: 'Reference',
      collapsed: true,
      autogenerate: { directory: 'reference' },
    },

    // Manual items
    {
      label: 'Getting Started',
      items: [
        { label: 'Introduction', slug: 'intro' },
        { label: 'Installation', slug: 'install' },
        { label: 'External Link', link: 'https://example.com', attrs: { target: '_blank' } },
      ],
    },

    // Badge on items
    {
      label: 'Advanced',
      items: [
        { label: 'Plugins', slug: 'advanced/plugins', badge: { text: 'New', variant: 'tip' } },
        { label: 'Deprecated', slug: 'advanced/old', badge: { text: 'Deprecated', variant: 'danger' } },
      ],
    },
  ],
})
```

## Frontmatter for Doc Pages

```markdown
---
title: My Page Title
description: SEO description (max 160 chars)
sidebar:
  label: Short Label   # Override sidebar label
  order: 2             # Sort order within autogenerated sections
  badge:
    text: New
    variant: tip       # tip | note | danger | caution | success | default
  hidden: false        # Hide from sidebar but keep accessible
head:
  - tag: meta
    attrs: { name: robots, content: noindex }
tableOfContents: false  # Disable ToC for this page
editUrl: false          # Disable edit link for this page
template: splash        # Use splash layout (landing page)
hero:
  title: Welcome
  tagline: Build faster docs
  image:
    file: ../../assets/hero.webp
  actions:
    - text: Get Started
      link: /intro/
      icon: right-arrow
      variant: primary
    - text: GitHub
      link: https://github.com/org/repo
      icon: github
      variant: minimal
---
```

## Internationalization (i18n)

```typescript
// astro.config.mjs
starlight({
  defaultLocale: 'en',
  locales: {
    en: { label: 'English', lang: 'en' },
    uk: { label: 'Українська', lang: 'uk' },
    fr: { label: 'Français', lang: 'fr' },
  },
})
```

**File structure for i18n:**

```
src/content/docs/
├── en/
│   ├── intro.md
│   └── guides/
│       └── first-steps.md
├── uk/
│   ├── intro.md
│   └── guides/
│       └── first-steps.md
└── fr/
    └── intro.md
```

**Accessing translations in components:**

```astro
---
import { useTranslations } from '@astrojs/starlight/utils/translations';
const t = useTranslations(Astro.currentLocale ?? 'en');
---
<p>{t('page.editLink')}</p>
```

## Custom Components (Overrides)

Replace any Starlight built-in component:

```typescript
// astro.config.mjs
starlight({
  components: {
    Header: './src/components/CustomHeader.astro',
    Footer: './src/components/CustomFooter.astro',
    Hero: './src/components/CustomHero.astro',
    Sidebar: './src/components/CustomSidebar.astro',
    ThemeSelect: './src/components/CustomThemeSelect.astro',
    SocialIcons: './src/components/CustomSocialIcons.astro',
    TableOfContents: './src/components/CustomToC.astro',
    Pagination: './src/components/CustomPagination.astro',
  },
})
```

**Extending (not replacing) a component:**

```astro
---
// src/components/CustomHeader.astro
import Default from '@astrojs/starlight/components/Header.astro';
---

<Default {...Astro.props}>
  <div slot="before-title" class="badge">Beta</div>
</Default>
```

## Deployment

```yaml
# .github/workflows/deploy.yml
name: Deploy Starlight to GitHub Pages
on:
  push:
    branches: [main]
permissions:
  contents: read
  pages: write
  id-token: write
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: npm
      - run: npm ci
      - run: npm run build
      - uses: actions/upload-pages-artifact@v3
        with:
          path: dist/
  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    steps:
      - id: deployment
        uses: actions/deploy-pages@v4
```

## Search Configuration

```typescript
// Default: Pagefind (static, no server required)
starlight({
  pagefind: true, // default

  // Disable if using external search (Algolia DocSearch, etc.)
  pagefind: false,
  head: [
    { tag: 'link', attrs: { rel: 'stylesheet', href: 'https://cdn.jsdelivr.net/npm/@docsearch/css@3' } },
    { tag: 'script', attrs: { src: 'https://cdn.jsdelivr.net/npm/@docsearch/js@3' } },
  ],
})
```

## Common Issues

| Issue | Cause | Fix |
|-------|-------|-----|
| Page not in sidebar | Missing from `sidebar` config or not in autogenerated directory | Check file path and `sidebar` config |
| i18n page not found | Missing translation file | Create matching file in locale directory |
| Custom component not rendering | Wrong import path in `components` override | Use `./src/...` relative to project root |
| Style not applied | CSS specificity issue | Add `[data-page-id]` selector or use `!important` |
| Build fails on images | Remote image domain not allowed | Add to `image.domains` in Astro config |
